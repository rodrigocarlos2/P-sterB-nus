# Tópicos

<h2> Mobile Computing </h2>

<p align="justify">

Caracteriza-se  como uma tecnologia que permite a 
transmissão  de dados, voz e de vídeo através de um 
computador ou qualquer outro dispositivo sem que haja a necessidade  de uma ligação física e fixa. 

</p>

<p align="justify">

A Mobile Computing engloba dois conceitos principais que são: Hardware Móvel e Software Móvel. Hardware Móvel inclui dispositivos e ou componentes de um dispositivo que recebem serviços de portabilidade. Esses dispositivos podem ser laptops, smartphones, tablets e PC portáteis. 

</p>

<p align="justify">

O Software Móvel é o programa executado no Hardware Móvel. Eles podem ser Sistemas Operacionais como Android, Windows Mobile, IOS, entre outros.  

</p>

</p>

<h2> Cloud Computing </h2>

<p align="justify">

Cloud Computing (Computação em Nuvem) é a entrega da computação como um serviço ao invés de um produto, onde recursos compartilhados, software e informações são fornecidas, permitindo o acesso através de qualquer computador, tablet ou celular conectado à Internet. 

</p>

<p align="justify">

Exemplos de Cloud Computing são os serviços como Dropbox, Google drive, iCloud e Onedrive, onde os usuários podem criar e editar documentos online e sincronizar arquivos ao mesmo tempo, com única restrição de conexão com a Internet.

</p>

<p align="center">
  <img src="CloudComputing.jpg" width="60%">
</p>

<h2> Mobile Cloud Computing </h2>

<p align="justify">

O Mobile Cloud Computing (MCC) é a combinação de Cloud Computing com Mobile Computing, com o objetivo de trazer mais recursos computacionais para usuários móveis, operadores de rede e provedores de Cloud Computing. 

</p>

<p align="justify">

O objetivo geral do MCC é permitir a execução de aplicativos móveis em larga escala de dispositivos móveis, com uma melhor experiência do usuário final.

</p>

<p align="justify">

Dentre as vantagens do MCC estão a utilização de menos recursos locais dos dispositivos, suporte a uma grande quantidade de dispositivos e maior confiabilidade das informações já que estão protegidas e armazenadas na nuvem.

</p>

</p>

<h2> Internet of Things </h2>

<p align="justify">

Internet of Things (IoT) é um conceito tecnológico em que todos os objetos da vida cotidiana estão conectados à internet de modo inteligente e/ou sensorial.

</p>

<p align="justify">

A IoT consiste na ideia da fusão do mundo real com o mundo digital, fazendo com que o indivíduo possa estar em constante comunicação e interação, seja com outras pessoas ou objetos.

</p>

<p align="justify">

São muitos os os objetos que podem ser conectados plea IoT como, por exemplo, eletrodomésticos, carros, casas, wearables (dispositivos vestíveis), etc.

</p>
